# 🧠 Student Mental Health Analysis (EDA)
## 📌 Overview
This project performs **Exploratory Data Analysis (EDA)** on a Student Mental Health dataset using Python. The objective is to identify patterns, trends, and relationships among factors such as study hours, stress levels, sleep quality, heart rate, and physical activity.
The project includes data cleaning, statistical analysis, and visualizations to generate meaningful insights into student well-being.

## 🎯 Objectives
- Clean and preprocess the dataset
- Analyze the distribution of important variables
- Explore relationships between academic and lifestyle factors
- Identify trends, correlations, and outliers
- Generate meaningful insights using visualizations

## 🛠️ Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Dataset
The dataset contains information related to student mental health, including:
- Study Hours
- Stress Levels
- Sleep Quality
- Heart Rate
- Physical Activity
- Gender
- Other lifestyle-related attributes

## 📊 Project Workflow
1. Import Libraries
2. Load Dataset
3. Data Exploration
4. Data Cleaning
5. Handling Missing Values
6. Removing Duplicates
7. Finding Outliers
8. Visualization
9. Key Insights
10. Conclusion

## 📈 Visualizations
The project includes the following visualizations:
- Histogram
- Count Plot
- Box Plot
- Scatter Plot
- Bar Plot

## 🔑 Key Insights
- Most students fall within moderate study hours and stress levels.
- Sleep quality tends to decrease as stress levels increase.
- Heart rate contains a few outliers.
- Study hours alone do not strongly predict stress levels.
- Correlation analysis highlights relationships among lifestyle and mental health factors.

## 📌 Conclusion
This project demonstrates how Exploratory Data Analysis (EDA) can transform raw data into meaningful insights. Through data cleaning and visualization, the analysis highlights patterns related to student mental health and provides a strong foundation for future predictive analytics and machine learning applications.
